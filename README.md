# Projetos de Estudos

Colecao de projetos praticos desenvolvidos para treinar **HTML e CSS**, reunindo exercicios de interface, layout e estruturacao de paginas.

## Visao Geral

Este repositorio funciona como um espaco de estudos, reunindo pequenos projetos criados ao longo da pratica com frontend. A proposta aqui e mostrar evolucao, repeticao e consolidacao de fundamentos.

## Projetos Incluidos

### `DESAFIOCSS1`

Landing page de estudo com foco em layout, tipografia, espacamento e composicao visual.

### `LOGIN`

Tela de login simples para praticar formularios, campos de entrada e estrutura HTML.

### `site-roupas-de-noivas`

Estrutura inicial de uma pagina para estudo de composicao e organizacao basica de arquivos.

## Tecnologias

- HTML5
- CSS3

## Como Executar

1. Clone este repositorio:

```bash
git clone https://github.com/victorbernardo18-oss/PROJETOS-ESTUDOSGITS.git
```

2. Entre na pasta do projeto:

```bash
cd PROJETOS-ESTUDOSGITS
```

3. Abra o `index.html` do projeto que deseja visualizar:

- `DESAFIOCSS1/index.html`
- `LOGIN/index.html`
- `site-roupas-de-noivas/index.html`
